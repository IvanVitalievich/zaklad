# Закладки / Not Bookmarks

## 📌 Описание
Учебный проект с серверной частью, реализующий систему регистрации, входа и взаимодействия пользователей с контентом.

Проект выполнен в рамках курса (дипломная работа).

## ⚙️ Функционал
- Регистрация пользователей
- Авторизация (login/logout)
- Всплывающие уведомления (alerts / notifications)
- Серверная логика (backend)
- Шутливый интерфейс с концепцией "закладки / незакладки"

## Технологии
-JavaScript(Vanilla)
-Node.js
-HTML
-CSS

## Статус проекта
Backend не задеплоен, проект работает локально.

## Примечание
Проект учебный, создавался в процессе обучения и отладки серверной логики.

## Как запустить проект

### 1. Установка зависимостей
npm install
2. Запуск сервера
node server.js

или

npm start
3. База данных

Проект использует SQLite (db.sqlite), подключение происходит автоматически при запуске сервера.
