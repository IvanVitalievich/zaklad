# Закладки / Not Bookmarks

## 📌 Описание
Учебный проект с серверной частью, реализующий систему регистрации, входа и взаимодействия пользователей с контентом.

Проект выполнен в рамках курса (дипломная работа).

  ## Возможности проекта

- Регистрация и авторизация пользователей
- Работа с сессиями / логикой входа
- Всплывающие уведомления (alerts / notifications)
- Взаимодействие с базой данных SQLite
- Клиент-серверная логика на Node.js
- Учебный проект с элементами реального full-stack приложения

## Технологии
- JavaScript(Vanilla)
- Node.js
- HTML
- CSS

## Статус проекта
Backend не задеплоен, проект работает локально.

## Примечание
Проект учебный, создавался в процессе обучения и отладки серверной логики.

## Как запустить проект

### 1. Установка зависимостей
npm install
2. Запуск сервера
node server.js

или

npm start
3. База данных

Проект использует SQLite (db.sqlite), подключение происходит автоматически при запуске сервера.

📌 Примечание

Проект запускался локально в рамках обучения и дипломной работы.
